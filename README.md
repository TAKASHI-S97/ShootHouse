# ShootHouse（現在開発中）

## 概要

C#（WinForms）で開発中の **2D 見下ろし型ミニシューティングゲーム**。  
「技術のプロモーション」兼「暇つぶし開発」で制作しています。

敵を倒してステージ内のフラグをアクティブにし、ステージをクリアしながら進んでいく、  
シンプルながら成長要素のあるミニゲームです。

---

## ゲーム仕様

### ■ 基本システム
- 2D 見下ろし型のアクションシューティング
- マップ内には **障害物** があり、
  - プレイヤー弾・敵弾は **貫通しない**
  - **破壊不可**

---

### ■ ステージシステム
- 各ステージには **クリア用フラグ** が存在
- フラグは初期状態では無効
- 敵を **一定数撃破するとアクティブ** になり、到達するとステージクリア
- ステージクリア後は  
  **次ステージへ進む** / **ゲームを終了する** の選択が可能

---

### ■ 敵
- ステージが進むと **敵の人数が増加**
- そのステージでの **撃破数によって同時出現数も増加**
- 敵の攻撃力・体力は **ステージが上がるごとに強化**

---

### ■ プレイヤー
- ステージクリア時に **体力全回復**
- **大技（1回使用可能）** を所持  
  → 発動すると **画面内のすべての敵を即撃破**
- アイテムにより大技回数を増加可能

---

## ショップシステム
- ステージクリア後に **ショップが出現**
- ランダムに選ばれた **3つのアイテムから1つを無料で選択**
- **無傷クリア** の場合、選択肢から **2つ** 選べるようになる

### アイテム一覧
- **体力アップ**
- **攻撃力アップ**
- **移動速度アップ**
- **大技使用回数 +1**

---

## 操作方法

| 操作 | 内容 |
|------|------|
| WASD | 移動 |
| マウス | 照準 |
| 左クリック | 通常ショット |
| 右クリック | 大技 |
| Esc | ポーズメニュー |

---

## 技術要素 / 実装内容

- **C#（.NET 9）+ WinForms**
- `Timer` によるなんちゃって FPS 管理
- プレイヤー画像は初期化時に **ビットマップへバッファ化**
- 回転描画は `DrawImage(Image, PointF[])` の **3点指定オーバーロード**を使用  
  → **Graphics.Transform を汚さずに回転可能**
- `.cur` カーソルファイルを使用した **カスタム照準カーソル**
- 弾オブジェクトの **リスト管理**
- 抽象クラスによる **類似クラス共通化**
